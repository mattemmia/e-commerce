## ✨ Key Features

### **For Customers**
- **Product Catalog**: Browse goods with images, prices, and descriptions from Firebase
- **Add/Remove Cart**: Smooth cart management with quantity updates
- **Customer Ratings**: Rate and review products - ratings stored in Firebase
- **Real-time WhatsApp Ordering**: Click to order and chat with admin instantly

### **For Admin**
- **Admin Login Page**: Secure Firebase Authentication
- **Inventory Management**: Real-time stock updates with Firestore
- **Sales Analytics**: View selling rate, top products, and total revenue
- **Order Tracking**: Track goods from order to delivery in real-time
